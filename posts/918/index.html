Problem:

P918 Maximum Sum of Circular Subarray

https://leetcode.cn/problems/maximum-sum-circular-subarray/description/?envType=study-plan-v2&envId=top-interview-150

It can be divided into two cases:

  • The first is the classic Kadane’s algorithm pattern, subarray [start, end], 0<=start<=end<n 0 <= start <= end < n 0<=start<=end<n
  • The second case, the subarray is divided into [0, end] and [start, n-1], 0<=end<=start<n 0 <= end <= start < n 0<=end<=start<n

For the second case, the sum can be obtained by total - sum(end, start) of the entire array. Therefore, we need to find the minimum sum of the subarray.

Note that in the second case, the subarray length cannot be the entire array length, in which case total - sum == 0
For example: nums = [-3,-2,-3]
