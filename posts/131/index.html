P131 Palindrome Partitioning

Backtracking: Find combinations of string sizes for partitioning, where each combination is a palindrome

However, the above method for checking palindromes involves a lot of repeated calculations, so consider dynamic programming

We can preprocess whether each substring s[i…j] of string s is a palindrome, using dynamic programming. Let f(i,j) represent whether s[i…j] is a palindrome, then the state transition equation is:

f(i,j)=True,i≥jf(i,j) = True, i≥j f(i,j)=True,i≥j

f(i,j)=f(i+1,j−1)∧(s[i]=s[j])f(i,j) = f(i+1,j−1) ∧ (s[i]=s[j]) f(i,j)=f(i+1,j−1)∧(s[i]=s[j])
